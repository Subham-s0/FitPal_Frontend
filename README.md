# FitPal

## Project Info

FitPal is your ultimate fitness companion application.

## Getting Started

To run the project locally:

1.  Clone the repository.
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```

## Technologies Used

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS
